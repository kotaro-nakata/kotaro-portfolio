# Design System & Log

Kotaro Nakata ポートフォリオのデザインルールと変更履歴をまとめたドキュメント。
新しいセクションやコンポーネントを足すときは、このルールに沿うことで統一感を保つ。

> **現行テーマ: 手描きZINE / 研究ノート風**（2026-06-14〜）。
> 紙・インク・ベタ塗り・手描き装飾が基調。SaaS的な均一カードUIやグラデ多用は避ける。

---

## 1. デザイン原則

- **手描きZINE / 研究ノート感** — 紙っぽいオフホワイト背景、インクの線、ベタ塗り。少しラフで非対称、知的だが堅すぎない。
- **ベタ塗り中心・影は最小** — グラデーションは使わない。立体影の代わりに「ステッカー風のハードオフセット影」をホバー時だけ使う。
- **色数を絞る** — グリーン＝見出し・線・要点 / ピンク＝装飾の差し色 / ブラウン＝サブ。背景は紙、文字はインク黒。
- **手描き装飾は散らすが可読性は守る** — 波線下線・手描き丸・破線・矢印・回転は効かせるが、本文の読みやすさは犠牲にしない。
- **余白を大きく** — 1セクション＝冊子の1ページのように、たっぷりの余白で見せる。
- **避けること** — 均一なカードUI / グラデ多用 / 黒背景の近未来UI / きれいすぎるアイコン / 過度なガラスモーフィズム / 完全な左右対称 / Bootstrapっぽい無難さ。
- **可読性優先** — 極端に小さい文字をつくらない（最小 `--fs-xs` = 0.78rem）。本文 16.5px / 行間 1.78。
- **アクセシビリティ** — インタラクティブ要素には `aria-*`。`prefers-reduced-motion` を尊重。

## 3. タイポグラフィ

**見出しは手描きフォント・本文は読みやすいサンセリフ。** コントラストを大きく。

- 見出しフォント: `--font-hand` = `"Yusei Magic", "Inter", "Noto Sans JP", sans-serif`（手描き・少し歪んだ・太め）
  - 適用先: `h1` / `h2` / `.brand` / `.eyebrow` / `.heading-sub` / `.chapter-period` / `.service-card h3`
  - ウェイトは 400（フォント自体が太い）。`letter-spacing` は 0。
  - 見出しは少し回転させる（`rotate(-1〜-1.8deg)`）。完全な水平を避ける。
- 本文フォント: `"Inter", "Noto Sans JP", system-ui, sans-serif`（weight 400、行間広め）
- 見出しは**大胆に大きく**、本文は**小さめ**。両者のコントラストを強くとる。

| 変数 | 値 | 用途 |
|---|---|---|
| `--fs-hero` | `clamp(2.7rem, 8.5vw, 5.4rem)` | ヒーロー H1 |
| `--fs-h2` | `clamp(2rem, 5.5vw, 3.4rem)` | セクション見出し |
| `--fs-h3` | `1.08rem` | カード見出し |
| `--fs-lead` | `clamp(1.02rem, 2vw, 1.25rem)` | リード文・キャッチ |
| `--fs-body` | `0.96rem` | 本文 |
| `--fs-sm` | `0.87rem` | カード説明・リスト |
| `--fs-xs` | `0.78rem` | 日付・タグ（最小） |
| `--fs-eyebrow` | `0.8rem` | （アイブロウは手描き体で約1rem表示） |

- セクション見出しは単語1つで統一: **Works / Projects / Services / Experience**。
- ヒーロー・キャッチは中央寄せ（`.hero-minimal { text-align: center }`）。

## 4. 背景

- **静的な紙**。`--bg` のオフホワイト＋うっすらした研究ノート風のドットグリッド。
  ```css
  background-color: var(--bg);
  background-image: radial-gradient(rgba(58,42,36,0.05) 1px, transparent 1.4px);
  background-size: 22px 22px;
  background-position: -11px -11px;
  ```
- ドットは極薄（opacity .05）。罫線感を出すが本文の邪魔をしない。
- `.section.alt` は上下フェードする薄いグリーンのティントで差をつける（帯の継ぎ目はつくらない）。
- ※旧テーマの「3つの光球が動く動的背景」は廃止（`main.js` の追従コードは残るが背景には未使用）。

---

## 5. 手描き装飾

- **セクション見出しの下線** — グリーンの手描き波線（インラインSVGを `::after` の背景に）。`.simple-heading h2` / `.section-heading > h2`。
- **ヒーロー名の下線** — 同じ波線をピンクで（`.hero-minimal h1::after`、stroke `#e84a9b`）。
- **ナビのホバー** — リンク下にピンクの波線がにじむ（`.nav-links a:hover::after`）。
- **アイブロウ／サブ見出し** — 手描き体で少し回転（ピンク or ブラウン）。注釈・メモのように点在させる。
- **タイムライン** — 破線のノート罫線＋丸マーカー。転換点（`.highlight`）はピンクのドット。
- 手描きSVGは `stroke="#1e1e1e"`・`stroke-linecap/linejoin="round"` のインク線で。完璧なアイコンよりラフなスケッチ感を優先。

## 6. コンポーネント規約

- **カード**（`.work-card` / `.aside-item` / `.service-card` / `.skill-card` / `.contact-card`）:
  `--card` 面 + 細いインク枠（`1〜1.5px var(--text)`）+ `--radius`(3px) + 影なし。
  ホバーで `translate(-3px,-3px)` + `--shadow-hover`（ステッカー風にカチッとずれる）。
  サービスカードは `:nth-child` で交互に微回転（±0.6〜0.7deg）させ、手置き感を出す。量産しない。
- **ボタン**（`.button`）: `2px solid var(--text)` 枠 + 紙色塗り + `--radius`。`min-height: 44px`。
  ホバーでピンク塗り＋白文字＋わずかに回転＆オフセット影。`primary` は通常グリーン塗り→ホバーでピンク。
- **チップ／タグ**（`.skill-tags` など）: 既存の丸タグ。塗りは緑地に白、枠はミュート色。
- **モーダル/詳細ビュー**: 角丸 `--radius`、スマホでは画面下から出るシート型。オーバーレイは `rgba(30,30,30,.5)`。
- **ナビ**: 紙背景＋下に `2px var(--text)` のインク罫線。ブランドはグリーンの手描き体。
  デスクトップ横並び、**820px以下でハンバーガー**（下ドロップ式・×印アニメ・自動クローズ・`aria-expanded` 連動）。

---

## 7. アニメーション

- 派手にしない。手描き線が少し伸びる／ホバーで少しズレる／スクロールで紙面が自然に現れる程度。
- 3D演出や過剰なモーションは使わない。

---

## 8. レスポンシブ

- `clamp()` の無段階スケーリング + メディアクエリ併用。
- ブレークポイント:
  - **≤820px** — ナビをハンバーガー化、Services/Skills を縦リスト型、Contact を1カラム。
  - **≤640px** — カードグリッドの最小幅を縮小、間隔を詰める。
  - **≤560px** — 全グリッド1カラム、セクション余白縮小、モーダルをシート型に。
- `<meta name="viewport">` 設定済み。画像は `max-width: 100%`。横スライダーは `scroll-snap`。

## 9. 変更履歴（Log）

### 2026-06-14 — 手描きZINE / 研究ノート風に全面リデザイン
- **パレット刷新**: 紙オフホワイト `#f7f3ea` ＋ グリーン `#159b4f` / ピンク `#e84a9b` / ブラウン `#3a2a24`。ベタ塗り中心、影とグラデを撤去（ホバーのみハードオフセット影）。
- **手描きフォント**: 見出し・ブランド・年号・サービス名に **Yusei Magic**。本文はサンセリフのまま小さめ・行間広めでコントラストを強調。見出しは少し回転。
- **手描き装飾**: セクション見出しにグリーンの波線下線、ヒーロー名にピンクの波線下線、ナビホバーにピンク波線、アイブロウ／サブ見出しは回転した手書きメモ風。
- **背景**: 動く光球を廃止し、静的な紙＋研究ノート風ドットグリッドに。
- **カード／ボタン**: 角丸最小・細いインク枠・影なし。ホバーでステッカー風にずれる。サービスカードは交互に微回転。ボタンは枠線＋紙色→ホバーでピンク。
- **タイムライン**: 破線のノート罫線＋丸マーカー、転換点はピンクのドット。
- **サービス図解**: アイソメ3D図を手描き風スケッチ（インク輪郭＋頂点ジッター、新パレット）で再生成。
- **Services セクション**: 旧 Skills を「Services / ご依頼いただけること」に作り替え（受託Web/SaaS・海外展開・ハード/IoT・EC/AI の4枚、図解＋一文、技術タグ無し、案件は匿名）。
